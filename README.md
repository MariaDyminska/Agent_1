FrozenLake з AI-агентом (Q-learning)
           Мета роботи

Метою даної роботи є розробка гри FrozenLake та реалізація AI-агента, який навчається проходити рівень за допомогою алгоритму навчання з підкріпленням (Q-learning).

            Опис гри

FrozenLake — це grid-середовище, у якому агент:

починає рух із клітинки Start (S),

повинен дістатися до Goal (G),

уникаючи клітинок Hole (H).

Особливістю гри є те, що поверхня є слизькою, тому дії агента можуть виконуватись не завжди точно.

Приклад карти:

S F F F
F H F H
F F F H
H F F G

          Опис AI-агента

Для навчання агента було використано алгоритм Q-learning, який зберігає значення корисності дій у вигляді Q-таблиці.

Основні особливості реалізації:

використовується стратегія ε-greedy (дослідження + використання знань);

додано штраф за кожен крок, щоб агент не робив зайвих дій;

підсилена нагорода за досягнення цілі;

агент навчається в стохастичному середовищі (is_slippery = True).

У результаті агент досягає ≈80–90% успішних проходжень.

          Використані технології

Python

бібліотека gymnasium

бібліотека numpy

          Запуск програми

Для запуску необхідно виконати команду:

python frozenlake_ai.py


Після завершення навчання програма:

виводить показник winrate,

демонструє гру агента у вікні середовища.

          Результат

У ході виконання роботи було створено AI-агента, який:

навчається знаходити оптимальний шлях;

адаптується до випадковості рухів;

демонструє стабільну кількість перемог у грі FrozenLake.

        Висновок

У процесі виконання роботи я ознайомилася з принципами reinforcement learning, реалізувала алгоритм Q-learning та застосувала його для розв’язання задачі керування агентом у середовищі FrozenLake.
